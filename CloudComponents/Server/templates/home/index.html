As part of this project I want to create a security system which visualizes live camera feed and displays them. Those cameras can be standard esp32cam microcontrollers or can be small robot cars, capable of movement. Each device can be programmed separately, you can customize each one with the help of a terminal. You can also turn on and off features such as person detection and lane assist for each device. Person detection is the analysis of the footage and whenever it detects a person it notifies you on your phone, using a mobile application. lane assist helps the robot cars move. A robot car is powered by 4 Li-ion batteries and uses 2 microcontrollers (Arduino Uno & ESP32Cam) and a motor driver module (L298N).



There is also a desktop application that allows you to record the feed from the devices so that you can replay it later.
